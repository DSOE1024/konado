# Konadotnet - Konado .NET API

## 简介

Konado的.NET API，用于在Godot C#项目中使用Konado API来控制Konado对话，该项目是Konado项目的一部分。

## 依赖

- Godot 4.4.1 C# 版本
- .NET 8.0 SDK
- Konado

## 使用方法

请先启用Konado插件，然后再启用Konado .NET API插件。
