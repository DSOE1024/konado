## 簡介

Konado 是一個視覺小說引擎框架，提供多種對話模板以及對話管理器，可以快速創建對話遊戲，也可以嵌入各類遊戲的對話場景。


## 快速開始

請參考 [快速開始](https://godothub.com/konado/tutorial/install.html) 文件，了解如何快速開始使用 Konado。
