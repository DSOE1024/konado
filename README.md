## 简介

Konado是一个视觉小说引擎框架，提供多种对话模板以及对话管理器，可以快速创建对话游戏，也可以嵌入各类游戏的对话场景。


## 快速开始

请参考 [快速开始](https://godothub.com/konado/quickstart) 文档，了解如何快速开始使用Konado。
